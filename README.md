# Name of the project

Portfolio website

## What is this?

It is site that showcases my technical skills and job related information

## Tech

- React + vite
- Tailwind css
- Eslint and prettier

## Setup

1. `git clone https://github.com/RushiRuns/Portfolio-site.git`
2. `cd taskflow && npm install`
3. `npm run dev` (starts on http://localhost:5173)

## Folder

- `src/assets` - images and icons
- `src/components` - reusable UI

## Features (MVP)

- Downloadable resume
- Clean and minimalistic design
- Responsive layout

## Run locally

1. git clone ...
2. npm install
3. npm run dev
